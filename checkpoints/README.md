# Checkpoints

This directory contains pre-trained model weights and baseline checkpoints for the project.

## Directory Contents

- `Deepr/`: Pre-trained weights for the Deepr model baseline.
- `StageNet/`: Pre-trained weights for the StageNet model baseline.

## Usage

Checkpoints are typically loaded by the training scripts in the `trainers/` directory or when running `main.py` with specific configuration settings.
