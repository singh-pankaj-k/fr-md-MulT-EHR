# References

This directory contains external documentation, scripts, and baseline implementations used to support the project.

## Directory Contents

- `PyHealth-master/`: A local copy or submodule of the PyHealth library, which is the foundational package for the project.
- `intro_to_mimic_db.pdf`: A reference document describing the MIMIC database structure.
- `scripts/`: Supporting shell scripts for environment setup or data management.

## Purpose

These references provide the necessary context and foundational code to build, understand, and benchmark the multi-task heterogeneous GNN models.
